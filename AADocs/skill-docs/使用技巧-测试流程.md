## 4, 测试概念
```
为了验证接口或者程序的功能，以期接口或者程序是否符合需求。

简单分解为白盒测试和黑盒测试

白盒测试：
测试对象是源程序，以程序内部的逻辑为基础的测试。
1，逻辑覆盖。
    1，语句覆盖。
    2，判定覆盖。
    3，条件覆盖。
    4，判定条件覆盖。
    5，条件组合覆盖。
    6，路径覆盖等等。
2，循环覆盖。
4，基本路径测试。


黑盒测试：
1，等价类划分：
    1，划分等价类，
    2，确定测试用例。
2，因果图
3，错误判断
4，边界值分析
5，综合策略



```
